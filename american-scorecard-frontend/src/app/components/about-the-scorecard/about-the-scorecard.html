How To Use - American Scorecard

What is the American Scorecard?

The American Scorecard is intended to be a non-partisan approach to assessing the status of the country and our communities using analytics
based on metrics formed using both interdisciplinary and specialized approaches. This Scorecard can fulfill multiple purposes depending on
how it is used and leveraged. The primary purpose is to provide a platform for experts and people who work to better society in general
a place where they can directly represent their findings in a format that makes it simple and open/accessible to the general public.
The secondary purpose is to create a centralized location of these findings in a uniform format, such that competitiveness and anti-competitiveness
between different groups can be analyzed rigorously and prevent misinformation that would allow for groups and individuals to use deception to gain
advantages instead of actual work.

Metrics and the Language of Logic

Metrics are effectively any kind of mathematically determined number, meant to represent a given situation. What this equates to is, if
anything in reality can be described using logic, then it should be possible to create a solvable math equation for that thing. If an equation
is solvable in a way such that it becomes a single number after recieving all inputs/variables, then it can be treated as a metric for the situation
that the equation describes logically.
